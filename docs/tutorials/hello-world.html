Hello World with NexusCore

Get started with NexusCore by creating your first AI-powered text editor application.

Welcome to NexusCore, the AI-first text editor designed to enhance your coding experience with intelligent features. In this tutorial, we'll walk you through creating a simple "Hello World" application using NexusCore's powerful API.

ℹ️

Prerequisites

Before you begin, make sure you have the following installed:

  • Node.js 16+ and npm
  • Git
  • A modern web browser (Chrome, Firefox, Safari, or Edge)

Getting Started

Let's create a simple Hello World application with NexusCore. Follow these steps:

Step 1: Install NexusCore CLI

First, install the NexusCore CLI globally using npm:

Terminal 
npm install -g @nexuscore/cli

Verify the installation by running:

$ nexuscore --version
@nexuscore/cli v1.0.0

Step 2: Create a New Project

Create a new directory for your project and initialize it with NexusCore:

$ mkdir hello-nexuscore
$ cd hello-nexuscore
$ nexuscore init
Initializing new NexusCore project...
Installing dependencies...
Project created successfully!

This will create a new project with the following structure:

Project Structure 
hello-nexuscore/
├── node_modules/
├── public/
│   ├── index.html
│   └── favicon.ico
├── src/
│   ├── index.js
│   └── styles.css
├── package.json
└── README.md

Step 3: Create Your First Component

Open src/index.js and replace its content with:

src/index.js 
import { Editor } from '@nexuscore/editor';
import '@nexuscore/editor/dist/styles.css';

// Initialize the editor
const editor = new Editor({
    target: document.getElementById('app'),
    props: {
        content: '# Hello, NexusCore!\n\nStart typing here...',
        theme: 'light',
        showLineNumbers: true
    }
});

// Expose editor instance for development
window.editor = editor;

Step 4: Update the HTML

Open public/index.html and update it to include your app container:

public/index.html 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Hello NexusCore</title>
    <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600&display=swap" rel="stylesheet">
    <link rel="stylesheet" href="/src/styles.css">
</head>
<body>
    <div id="app"></div>
    <script type="module" src="/src/index.js"></script>
</body>
</html>

Step 5: Add Some Basic Styling

Update src/styles.css with some basic styles:

src/styles.css 
:root {
    --nexus-blue: #2563EB;
    --gray-50: #F8FAFC;
    --gray-100: #F1F5F9;
    --gray-200: #E2E8F0;
    --gray-900: #0F172A;
}

body {
    font-family: 'Inter', -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif;
    margin: 0;
    padding: 2rem;
    background-color: var(--gray-50);
    color: var(--gray-900);
    line-height: 1.5;
}

#app {
    max-width: 800px;
    margin: 0 auto;
    background: white;
    border-radius: 8px;
    box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
    overflow: hidden;
}

/* Customize the editor */
.nexus-editor {
    min-height: 300px;
    border: 1px solid var(--gray-200);
    border-radius: 6px;
}

.nexus-toolbar {
    border-bottom: 1px solid var(--gray-200);
    padding: 0.5rem;
}

Step 6: Start the Development Server

Run the development server with hot-reload:

$ npm run dev
> hello-nexuscore@1.0.0 dev
> nexuscore dev

  VITE v3.0.0  ready in 234 ms

  ➜  Local:   http://localhost:3000/
  ➜  Network: use --host to expose

Open http://localhost:3000 in your browser to see your app in action!

💡

Tip

Try typing in the editor and use the toolbar to format your text. The editor supports Markdown syntax and keyboard shortcuts for a better writing experience.
